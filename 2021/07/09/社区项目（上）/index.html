git工具使用

命令 含义
git init 初始化
git status 查看工程状态（需要先cd到工程路径下）
git add * 向git添加所有资源
git commit -m ‘Test1’ 提交到本地仓库并为项目起别名为Test1

IDEA中使用git：

在Settings-Version Control-Git中配置Path to Git executable为git可执行程序

选中项目，点击VCS-Import into Version Control-Create Git Repository

点击VCS-commit，选择项目的src目录和pom.xml进行提交，在commit message处输入信息对提交进行备注

事务管理

REQUIRED：支持当前事务（外部事务），如果不存在则创建新事务

REQUIRES_NEW：创建一个新事务，并且暂停当前事务（外部事务）

NESTED：如果当前存在事务（外部事务），则嵌套在该事务中执行（独立的提交和回滚），否则和REQUIRED一样
