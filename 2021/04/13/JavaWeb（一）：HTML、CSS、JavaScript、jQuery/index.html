1. HTML

  • B/S软件结构：Browser（浏览器）、Server（服务器）

    JavaSE是C/S结构：Client（客户机）、Server

HTML标签：

  1. 标签格式：<标签名>封装的数据</标签名>

  2. 标签名大小写不敏感

  3. 标签拥有自己的属性：

    基本属性：bgcolor=”red” 修改简单样式效果

    事件属性：onclick=”alert(‘你好！’)” 设置事件响应后的代码

  4. 单标签：<标签名/>；双标签：<标签名>封装的数据</标签名>

标签语法：
1. font标签
2. 特殊字符
显示结果 实体名称
< &lt;
> &gt;
（空格） &nbsp;

注：HTML会裁掉多余的空格（输入10个只保留1个）

3. 标题标签

标题标签从h1到h6，h1最大，h6最小

aline属性是对齐属性：left左对齐（默认），center居中，right右对齐

4. 超链接

网页中所有点击后可以跳转的内容都是超链接

5. 列表标签

ul表示无序列表，ol表示有序列表，type属性可以修改列表项前的符号。li是列表项

6. img标签

img是图片标签，用来显示图片

JavaSE中：

​ 相对路径：从工程名开始
​ 绝对路径：盘符:/目录/文件名

Web中：

​ 相对路径：.表示当前文件所在目录 ..表示当前文件上一级目录 文件名表示当前文件所在目录的文件，相当于./文件名（./可以省略）
​ 绝对路径：http://ip:port/工程名/资源路径

7. table标签
8. 表格的跨行跨列

colspan属性设置跨列；rowspan属性设置跨行

9. iframe框架标签（内嵌窗口）

iframe标签可以在页面上开辟一个小区域来显示一个单独页面

iframe和a标签组合使用的步骤：

  1. 在iframe标签中使用name属性定义一个名称

    1. 在a标签的target属性上设置iframe的name属性值
10. 表单标签
10.1form标签：表单
type 种类
type=”text” 文件输入框 value设置默认显示内容
type=”password” 密码输入框 value设置默认显示内容
type=”radio” 单选框 name属性进行分组，checked=”checked”表示默认选中
type=”checkbox” 复选框 checked=”checked”表示默认选中
type=”reset” 重置按钮 value属性修改按钮上的文本
type=”submit” 提交按钮 value属性修改按钮上的文本
type=”button” 按钮 value属性修改按钮上的文本
type=”file” 文件上传域
type=”hidden” 隐藏域 （上传某些信息到服务器，不需要用户参与）

​ action属性设置提交的服务器地址

​ method属性设置提交的方式GET（默认）或POST

表单提交时，数据没有发送给服务器的三种情况：

  1. 表单项没有name属性值

    1. 单选、复选（下拉列表中的option标签）都要添加value属性
    2. 表单不在form标签中

GET请求：

  1. 浏览器地址栏地址：action属性[ + ? + 请求参数]

    请求参数的格式是：name=value&name=value

  2. 不安全

  3. 有数据长度限制

POST请求：

  1. 浏览器地址栏中只有action属性值

    1. 相对于GET请求更安全
    2. 理论上没有数据长度的限制

select标签：下拉列表框

​ option标签是下拉列表框中的选项，selected=”selected”设置默认选中

textarea表示多行文本输入框，起始标签和结束标签中的内容是默认值

​ rows属性设置显示几行的高度；cols属性设置每行显示几个字符的宽度

2. CSS

语法规则：p{属性: 值; 属性: 值… }

​ 属性和值被冒号分开，由大括号包围，组成一个完整的样式声明，例如：p{color: blue}

​ 定义多个声明需要用分号隔开，最后一条声明的最后可以不加分号

CSS注释：/*注释内容*/

1. CSS和HTML结合使用：

  1. 在标签的style属性上设置”key:value value;”，修改标签样式

    缺点：代码量庞大、可读性差、不可复用

  1. 在head标签中，使用style标签定义css样式

    缺点：只能在同一页面内复用代码，维护不方便

  1. 把css样式写成一个单独的css文件，再通过link标签引入

1. 标签名选择器

格式：标签名{属性: 值;}

标签名选择器可以决定哪些标签被动的使用这个样式

2. id选择器

id选择器格式：#属性值{属性: 值;}

通过id属性选择性的使用样式

3. class选择器（类选择器）

class选择器格式：.class属性值{属性: 值;}

4. 组合选择器

组合选择器格式：选择器1,选择器2,…,选择器n {属性: 值;}

组合选择器可以让多个选择器共用同一份css代码

3. JavaScript

JS是弱类型，Java是强类型

JavaScript特点：

  1. 交互性（信息动态交互）

    1. 安全性（不允许直接访问本地硬盘）
    2. 跨平台性（可以解释JS的浏览器都可以运行，与平台无关）

1.JavaScript和HTML结合使用

1. 在head或body标签中使用script标签书写JavaScript代码
2. 使用script标签引入单独的JavaScript代码文件

2. 变量

JavaScript变量类型：

数值类型 number
字符串类型 string
对象类型 object
布尔类型 boolean
函数类型 function

JavaScript中特殊值：

  1. undefined：未定义，所有js变量未赋值时，默认值都是undefined

    1. null：空值
    2. NAN：全称：Not a Number，非数字，非数值

JS中定义变量格式：

​ var 变量名; var 变量名 = 值;

1. 关系（比较）运算

等于：==，做字面值比较（不比较变量类型）

全等于：===，同时比较字面值和变量数据类型

2. 逻辑运算

在JavaScript语言中，所有的变量都可以当做boolean类型变量

0、null、undefined、””（空串）都可以认为是false

注：” “（有一个空格）不是空串，不是false

&&与运算：

  1. 当表达式全为真时，返回最后一个表达式的值
  2. 当表达式有一个为假时，返回第一个为假的表达式的值

||或运算：

  1. 当表达式全为假时，返回最后一个表达式的值
  2. 当表达式有一个为真时，返回第一个为真的表达式的值

&&与运算和||或运算有短路操作

3. 数组

数组定义方式：var 数组名= []; //空数组

var 数组名 = [1, “abc”, true]; //定义数组同时赋值

JavaScript中通过下标赋值时，会自动给数组做扩容操作

4. 函数

用function关键字定义函数

  • JavaScript中不允许函数重载，重载会覆盖上一次定义

  • 函数的arguments隐形参数（只在function函数内）

    隐形参数：在function函数中不需要定义，但可以获取所有参数的变量

    类似于Java中的可变长参数：public void fun(Object … args); 可变长参数是一个数组

5. JS中的自定义对象

​ Object形式的自定义对象

​ 大括号形式的自定义对象

6. JS中的事件

事件：电脑输入设备与页面进行交互的响应

常用的事件 应用
onload加载完成事件 常用于页面JS代码初始化操作
onclick单击事件 常用于按钮的单击响应操作
onblur失去焦点内容 常用于输入框失去焦点后验证输入内容是否合法
onchange内容发生改变事件 常用于下拉列表和输入框内容发生改变后操作
onsubmit表单提交事件 常用于表单提交前，验证所有表单项是否合法

事件的注册（绑定）：告诉浏览器事件响应后要执行哪些代码

事件的注册分为静态注册和动态注册：

  1. 静态注册事件：通过html标签的事件属性直接赋予事件响应后的代码
  2. 动态注册代码：先通过js代码得到标签的dom对象，通过dom对象.事件名 = function(){}赋予事件响应后的代码
onload加载完成事件
onclick单击事件
onblur失去焦点事件
onchange内容发生改变事件
onsubmit表单提交事件

4. JQuery

JQuery：JavaScript和Query，是辅助JavaScript开发的JS类库，解决浏览器兼容问题

JQuery中的$是一个函数

1. $：JQuery核心函数

$()就是调用$这个函数

  1. 传入参数为函数时：

    表示页面加载完成之后，相当于window.onload=function(){}

  1. 传入参数为HTML字符串时：

    会创建这些HTML标签对象

  1. 传入参数为选择器字符串时：

    $(“#id属性值”); id选择器

    $(“标签名”); 标签名选择器

    $(“.class属性值”); 类型选择器

  2. 传入参数为DOM对象时：

    会把dom对象转换为jquery对象

2. jQuery对象和dom对象

dom对象：

​ 通过getElementById()方法查询出的标签对象是dom对象

​ 通过getElementsByName()方法查询出的标签对象是dom对象

​ 通过getElementsByTagName()方法查询出的标签是dom对象

​ 通过createElement()方法创建的对象是dom对象

​ dom对象alert()出来的效果：[object HTML标签名Element]

jQuery对象：

​ 通过JQuery提供的API创建/查询的对象是JQuery对象

​ 通过JQuery包装的dom对象是JQuery对象

​ JQuery对象alert出来的效果：[object Object]

  • jQuery对象是dom对象的数组+jQuery提供的一系列功能函数

  • jQuery对象不能使用dom对象的属性和方法，dom对象也不能使用jQuery对象的属性和方法

dom对象转换为jQuery对象：$(dom对象)

jQuery对象转换为dom对象：jQuery对象[下标]取出相应的dom对象

基础选择器 含义
#id 根据id查找标签对象
.class 根据class查找标签对象
element 根据标签名查找标签对象
* 查找所有元素
selector1, selector2 组合选择器，合并选择器1、选择器2的结果并返回
  • p.myClass：表示标签名为p标签，且class类型为myClass的标签对象
层级选择器 含义
ancestor descendant 后代选择器，在给定祖先元素下匹配所有后代元素
parent > child 子元素选择器：在给定的父元素下匹配所有的子元素
prev + next 相邻元素选择器：匹配紧接在prev元素后的next元素
prev ~ sibings 之后的兄弟元素选择器：匹配prev元素后的所有siblings元素
基本过滤选择器 含义
:first 获取第一个元素
:last 获取最后一个元素
:not(selector) 去除所有与给定选择器匹配的元素
:even 匹配所有索引值为偶数的元素，从0开始计数
:odd 匹配所有索引值为奇数的元素，从0 开始计数
:eq(index) 匹配一个给定索引值的元素
:gt(index) 匹配所有大于给定索引值的元素
:lt(index) 匹配所有小于给定索引值的元素
:header 匹配所有如h1、h2、h3的标题元素
:animated 匹配所有正在执行动画效果的元素
内容过滤选择器 含义
:contains(text) 匹配包含给定文本的元素
:empty 匹配所有不包含子元素或文本的空元素
:parent 匹配含有子元素或文本的元素
:has(selector) 匹配含有选择器所匹配的元素的元素
属性过滤选择器 含义
[attribute] 匹配包含给定属性的元素
[attribute=value] 匹配给定属性值为value的元素
[attribute!=value] 匹配所有不含指定属性，或属性值不为value的元素
[attribute^=value] 匹配给定属性的值以value开头的元素
[attribute$=value] 匹配给定属性的值以value结尾的元素
[attribute*=value] 匹配给定属性的值包含value的元素
[attrSel1][attrSel2][attrSelN] 复合属性选择器，同时满足多个条件
表单过滤选择器 含义
:input 匹配所有inpute、textarea、select和button元素
:text 匹配所有文本输入框
:password 匹配所有密码输入框
:radio 匹配所有单选框
:checkbox 匹配所有复选框
:submit 匹配所有提交按钮
:reset 匹配所有重置按钮
:button 匹配所有input type=”button”
:file 匹配所有input type=”file” 文件上传
:hidden 匹配所有不可见元素 display:none 或 input type=hidden
表单对象属性过滤器 含义
:enabled 匹配所有可用元素
:disabled 匹配所有不可用元素
:checked 匹配所有选中的单选、复选、下拉列表的option标签对象
:selected 匹配所有选中的option
jQuery元素筛选 含义
eq() 获取给定索引的元素，功能与:eq()相同
first() 获取第一个元素，功能与:first相同
last() 获取最后一个元素，功能跟:last 一样
filter(exp) 留下匹配的元素
is(exp) 判断是否匹配给定的选择器，只要有一个匹配就返回true
has(exp) 返回包含有匹配选择器的元素的元素，功能与:has相同

children(exp) 返回匹配给定选择器的子元素，功能与parent>child相同
find(exp) 返回匹配给定选择器的后代元素，功能与ancestor descendant相同
next() 返回当前元素的下一个兄弟元素，功能与prev+next相同
nextAll() 返回当前元素后面所有的兄弟元素，功能与prev~siblings相同
nextUntil() 返回当前元素到指定匹配的元素为止的后面元素

4. jQuery属性操作

text()：设置/获取起始标签和结束标签中的文本，与dom属性innerText一样

val()：设置/获取表单项的value属性值，与dom属性value一样

attr()：设置/获取属性的值，不推荐操作checked、readOnly、selected、disabled等属性

prop()：设置/获取属性的值，只推荐操作checked、readOnly、selected、disabled等属性

8. jQuery事件操作

$(function(){})与window.onload = function(){}的区别：

  1. jQuery的页面加载完成之后是浏览器内核解析完页面标签创建好DOM对象后就立即执行
  2. 原生JS的页面加载完成之后，除了要等浏览器内核解析完标签创建好DOM对象，还要等标签显示时需要的内容加载完成

触发顺序：jQuery的页面加载完成后先执行；原生JS的页面加载完成后后执行

执行次数：原生JS的页面加载完成之后只执行一次赋值函数；jQuery的页面加载完成之后把全部注册的函数依次执行

click()：绑定/触发单击事件

mouseover()：鼠标移入事件

mouseout()：鼠标移出事件

bind()：给元素一次性绑定一个或多个事件

one()：使用上与bind()相同，但one()方法绑定的事件只响应一次

unbind()：与bind()方法相反的操作，解除事件绑定

live()：绑定事件，可以绑定所有与选择器匹配的事件（后来动态创建的事件也可以）

  • 事件的冒泡：父子援素同时监听同一个事件，当触发子元素事件时，该事件也被传递到父元素事件中进行相应

    阻止事件冒泡：在子元素函数体内return false

  • JavaScript事件对象

事件对象：是封装有触发的事件信息的一个JavaScript对象

获取JS事件对象：在事件的function(event)参数列表中添加一个参数event，这个event就是JavaScript传递事件处理函数的事件对象
